Objective

To configure a Wi-Fi router to function as a Wireless Access Point (WAP) for extending an existing network, while ensuring security and optimal performance.

Requirements

  • A Wi-Fi router (to be configured as WAP)
  • An active internet connection from the main router
  • Ethernet cable (UTP cable)
  • Laptop/PC for configuration
    • Criteria Excellent (5) Good (4) Fair (3) Poor (2) Needs Improvement (1)
    Accuracy of Configuration All settings correctly configured with no errors. Minor errors that do not affect functionality. Several errors but the setup is mostly functional. Significant errors affecting functionality. Setup is incorrect and non-functional.
    Understanding of Steps Clearly explains each step and its importance. Explains steps with minor misunderstandings. Some steps are unclear or missing explanations. Limited understanding of steps. Fails to explain steps correctly.
    Security Implementation Strong security settings applied (WPA3, MAC filtering, etc.). Security settings applied with minor issues. Some security settings applied but lacking key elements. Weak security settings implemented. No security settings applied.
    Testing & Troubleshooting Thorough testing with no connectivity issues. Minor issues identified and resolved. Some issues remain unresolved. Significant connectivity issues persist. No testing performed.

    Steps:

      1.Physical Setup:

    - Connect the main router's LAN port to the secondary router's LAN port using an Ethernet cable.

    - Ensure the secondary router is powered on.

      2.Access the Router Interface:

    - Connect your laptop/PC to the secondary router via Wi-Fi or Ethernet.

    - Open a web browser and enter the default IP address of the secondary router (e.g., 192.168.1.1).

    - Log in using the default username and password (usually printed on the back of the router).

      3.Disable DHCP on the Secondary Router:

    - Navigate to the "Local Network" or "LAN Settings" section.

    - Disable the DHCP server to ensure IP addresses are assigned by the main router.

      4.Assign a Static IP Address:

    - Assign a static IP address to the secondary router within the main router’s subnet but outside its DHCP

      5.Configure Wireless Settings:

    - Go to "Wireless Settings

    - Set the SSID (Wi-Fi name) and password identical to your main router for seamless roaming.

    - Choose a different channel than your main router to reduce interference.

      6.Enable Security Features:

    - Enable WPA3 encryption for enhanced security.

    - Optionally, configure MAC address filtering to allow only specific devices to connect.

      7.Save and Reboot:

    - Save all changes and reboot the secondary router.

    - Disconnect your laptop/PC from the secondary router.

      8.Test Connectivity:

    - Connect a device to the new WAP SSID.

    - Verify internet access by browsing or pinging an external IP like 8.8.8.8.

      9.Optional: Enable DHCP if Needed:

    - If you want the WAP to assign IP addresses, enable its DHCP server but ensure its IP pool doesn’t overlap with that of the main router.

      10.Verification:

    - Confirm that devices connected to both routers can communicate with each other and access the internet

    - Use traceroute (tracert on Windows) to verify that traffic passes through both routers correctly.
