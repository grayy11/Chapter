PRACTICE EXERCISE!

Instructions:

Follow these steps to troubleshoot your internet connection and answer the questions.

1. Check Your Internet Connection

  • Open a web browser and try to visit a website (e.g., YouTube or Wikipedia).

    • Question: Did the page load? (Yes / No)


    2. Check Your Device’s Connection Type

  • Look at how your device is connected to the internet.
  • If you're using Wi-Fi, check the signal strength (look for bars on your device).

    • Question 1: Are you connected to Wi-Fi or using an Ethernet cable? (Wi-Fi / Ethernet)

    Question 2 (If using Wi-Fi): Is the signal strength Strong / Weak?


    3. Perform a Basic Network Test

  • Open Command Prompt (Windows) or Terminal (Mac/Linux).
      • Type the following command and press Enter:
    • nginx
    • Copy code
    • ping 8.8.8.8
  • Check the results: If you see continuous responses, your connection is working. If you see "Request Timed Out," there might be an issue.

    • Question: Does the test show responses, or does it say "Request Timed Out"? (Responses / Timed Out)


    4. Restart Your Connection (If Needed)

  • If your internet is slow or not working, try these steps:

    • 1.Turn your Wi-Fi off and on (if using Wi-Fi).

    2.Restart your router/modem (unplug for 30 seconds, then plug it back in).

    3. Wait a few minutes and test your internet again.

  • Check the results: If you see continuous responses, your connection is working. If you see "Request Timed Out," there might be an issue.

    • Question: After restarting, did the connection improve? (Yes / No)
