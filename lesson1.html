Lesson 1: Install Network and Cables

LESSON 1: Install Networks and Cables


In this lesson, you will explore the fundamentals of network cable installation. A well-structured network is essential for seamless communication, and understanding how to install cables properly is a key skill in network management.


At the end of the Lesson, the Learners should be able to:

  • Identify the types of network cables and their uses.
  • Exhibit professionalism by following OH&S standards and 5S principles during cable installation.
  • Apply installation requirements and established procedures when installing network cables and cable raceways.
