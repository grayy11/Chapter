LESSON 2.2 IP Addressing


An IP address, or Internet Protocol address, is a unique string of numbers assigned to each device connected to a computer network that uses the Internet Protocol for communication. It serves as an identifier that allows devices to send and receive data over the network, ensuring that this data reaches the correct destination.


Parts of an IP Address:

Host Address
Network Address


A. IPv4 and IPv6

IP addresses can be classified in several ways based on their structure, purpose, and the type of network they are used in.

IPv4 (IP version 4): It is the current version of IP addresses (not for long). It is a 32-bit numeric address written as four numbers separated by periods.

Example: 66 . 94 . 234 . 13

Each group of numbers separated by periods is called an octet (66 - octet, 94 - octet, 234 - octet, 13 - octet). Number ranges from 0-255, and it can produce 4 billion possible unique addresses.

Computers and networks don’t read IP addresses in numeric format; they only understand numbers in binary format (1’s and 0’s).

For example:
66 = 01000010
94 = 01011110
234 = 11101010
13 = 00001101

66 . 94 . 234 . 13 = 01000010 . 01011110 . 11101010 . 00001101

Remember, byte is the singular form of bits. So:
1 Byte = 8 bits

B. IPv6 (IP version 6)
It is the next generation of IP addresses. It is a 128-bit hexadecimal address using both numbers and alphabets. With this version, it can produce over 340 undecillion addresses, more than enough for the foreseeable future.
Example: 76DC : 4F59 : 71CD : 9DC6 : 45CD : 67A2
In version 6, each hexadecimal character represents 4 bits.

Surface-Mounted Raceway

These are mounted directly onto the wall or surface, and they are typically used for retrofit applications where wires need to be added after construction.


If you are on a home/office private network and want to use TCP/IP, you should assign your computers/devices IP addresses from one of these three ranges. That way, your router/firewall would be the only device with a true IP address, making your network more secure.
