Router and WAP Configuration

L3.2 Router and WAP Configuration


Imagine a world without Wi-Fi—no streaming, no online gaming, and no instant access to information! Routers and Wireless Access Points (WAPs) are the backbone of seamless internet connectivity, ensuring that our devices stay connected anytime, anywhere.

In this lesson, you'll dive into the essentials of Router and WAP Configuration, learning how to set up, secure, and optimize network connections. Whether you're building a home network or managing a large-scale enterprise system, mastering these configurations is key to a smooth, efficient, and secure digital experience.

Ready to take control of your network? Let’s get started!

Wireless Network


A Wireless Network, commonly called WLAN or Wi-Fi, uses radio signal frequency to connect computers and network devices. This is based on the IEEE 802.11 standards rules and regulations.

What are the benefits of a Wireless Network?

  • Flexibility: With no wires, you can easily access your network resources from any location.
  • Cost: It reduces the cost of installation and labor.
  • Security: The wireless network uses a wireless router which has security features to encrypt your network traffic.

What are Wireless Routers?

It is a device that performs like a wired router without using a cable to connect to a network.

How to Configure a Wireless Router?

  • Plug in the AC Adapter and press the power button.
  • Connect one end of the networking cable to the router's Ethernet port and connect the other end to the LAN adapter of a PC or Laptop.
  • Access the router configuration panel through a web browser and on the URL tab type the IP address 192.168.0.1 or type http: tplinklogin.net (Note: This IP address is intended for TP link routers.)
  • On the login menu, enter the default username “admin” and the default password “admin”.
  • Once the username and password are verified and accepted the main panel of router configuration will appear, click on Wireless then choose Wireless Setting. Change the default Wireless Network name.
  • After changing and saving the Wireless Network name, it’s time to change the default password by clicking the Wireless Security Tab. Once the Wireless Security tab is clicked, another panel will appear. Change the Wireless Password, for example: 123456789 then Click Save.
  • Once the configuration was changed and saved, the device is required to reboot. Click the System Tool Tab on the lower part of the and click Reboot.
  • After the router configuration, search for the wireless network through the notification area. Connect the laptop or wireless device like a cellular phone from a wireless network, type in the password, then click connect.

    Configured DHCP and LAN IP of the router

    • On the main panel of the router configuration, select “Network” and click the LAN. Change the Default LAN IP from 192.168.0.1 to your desired LAN IP ex. 200.191.1.
    • On your computer, to refresh and detect the new default LAN IP, go CMD, type ipconfig/renew and enter to refresh and show the new IP configuration.
    • On the main panel of the router configuration, select “DHCP” and click the DHCP settings. Set the starting IP Address to 200.192.1.100 and Ending IP Address to 200.192.1.200, then click Save.

    MAC Address Authentication and Filtering

    • MAC address authentication is a method used for Network Admission Control (NAC). It controls user access rights based on access ports and user MAC address to protect security for enterprise network card is used to determine access to the network.
    • To assign specific IP address, get the MAC address of the PC. Go to CMD, type getmac.
    • On the main panel of the router configuration, select “Wireless’’ and click ‘’Wireless MAC Filtering’’. Select Add New and type the MAC Address of the wireless device you want to filter/block.
