Apply Security and Advanced Settings

L3.3 Apply Security and Advanced Settings

Wireless networks are essential for modern communication and internet access. However, they can be vulnerable to unauthorized access, data theft, and cyberattacks without proper security settings. This lesson will cover how to apply security and advanced settings in routers, Wi-Fi access points, and repeaters to ensure a safe and optimized network.

Here are some simple ways to secure your wifi router and secure your data.

1. Change the Default Router Login Credentials

Most Modems/Wi-Fi routers have their default username and Password, but some people don’t change it resulting in hackers and unauthorized connections to easily access your network. To configure Security/Firewall/Advanced settings for a router, Wi-Fi, wireless access point, or repeater, the following steps and options can be applied:

Here’s a step-by-step on how to configure your router:
  • Step 1: Type ipconfig on the command prompt or type CMD on the Windows search bar on your computer.
  • Step 2: Copy the default gateway and paste it on a browser.
  • Step 3: From there, set a username and password. Follow the instructions on the given pictures.

    • 2. Update the Router Firmware

    Firmware is software that is embedded into the device’s memory and responsible for its function. Router Manufacturers regularly release firmware updates to fix security vulnerabilities and improve performance. However, it is not designed to be updated frequently but outdated firmware can leave your Wi-Fi to vulnerable attacks.

    Here’s a step-by-step on how to update your Router Firmware:
  • Step 1: Check your router's firmware version by logging into the admin interface.
  • Step 2: Look for the “Firmware Update” or “Software Update” option.
  • Step 3: Follow the instructions to download and install the latest firmware.

    • 3. Use WPA3 Encryption

    Encryption is essential for protecting the data transmitted over your WiFi network. WPA3 (Wi-Fi Protected Access 3) is the latest and most secure encryption standard. If your router supports WPA3, you should enable it.

  • In the router’s admin interface, navigate to the wireless security settings.
  • Select WPA3 from the list of encryption options.
  • Save your changes.

    • If your router doesn’t support WPA3, use WPA2, which is still secure but slightly less so than WPA3. Avoid using older standards like WEP, as they are easily compromised.

    4. Create a Guest Network

    If you often have guests at your home who need internet access, it’s a good idea to create a separate guest network. This way, you can keep your primary network secure while still providing internet access to your visitors.

  • In the router’s wireless settings, look for the “Guest Network” option.
  • Enable the guest network and set a different password than your main network.
  • Configure the guest network to have limited access to your main network resources.

    • Parental Controls, Website Blocking, and Access Scheduling: Many routers offer parental control features to block certain types of traffic or specific websites. You may also be able to control when Internet access is disabled, even on a per-computer basis.

    Firewall: The document does not specifically discuss firewall settings, it is important to configure firewall settings to protect the network from unauthorized access.

    Advanced Settings

    Dynamic DNS (DDNS): If you are hosting a server, DDNS allows you to connect remotely using a fixed hostname (e.g., yourcomputer.service.com) even if your IP address changes.

    Port Forwarding, Port Triggering, DMZ & UPnP: These settings manage incoming traffic. Port forwarding directs traffic on specific ports to a specific computer. Port triggering automatically enables port forwarding when a program opens a connection on a certain port. A DMZ sends all incoming connections to a single computer. UPnP allows programs to forward their own ports, but it is less secure.

    Reboot the Router:After making changes, reboot the router for the settings to take effect . A reboot option is usually available on the configuration pages.

    Third-Party Router Firmwares:For advanced users, installing third-party firmware like DD-WRT, Tomato, or OpenWRT can provide additional options and capabilities.

    Importance of Securing a WiFi Network

    A secured wireless network helps to:

  • Prevent unauthorized access.
  • Protect sensitive data from cyber threats.
  • Improve network performance by restricting unknown users.
  • Avoid bandwidth theft that slows down the connection.
